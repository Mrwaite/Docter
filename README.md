##Docter

**病人看病流程**：

1. 注册并登录（此时可以加入个人信息的录入）
2. 进入心理测评
3. 选择一个题目开始做题
4. 完成后系统自动给予打分，并给出测评报告
5. 系统结合测评报告给予康复产品
6. 还可以进入咨询大厅咨询医生


**医生页面模块**：

**首页**：公告，我的客户，个人信息，咨询记录，预约测评，预约咨询，咨询大厅

**测评中心**：

**咨询中心**：展示咨询的一些概念或理念

**康复中心**：以列表的形式所有的康复资料

**报告中心**：以列表的形式显示所有的用户的预警信息

**预警中心**：审核报告，修改/查看，打印报告，以列表的形式显示所有用户的报告
